### Objetivos Principais:

- Como cientista de dados, criar uma proposta preditiva, ou como analista de dados realizar uma proposta analítica demonstrando o impacto da ONG Passos Mágicos.
- Na proposta analítica, deve-se criar um dashboard e storytelling.
- Na proposta preditiva, deve-se criar um modelo (Machine Learning, Deep Learning ou NLP) que preveja o comportamento do estudante, baseado em algumas variáveis disponibilizadas.
- Fazer um MVP (Mininum Viable Product) do modelo em produção utilizando o Streamlit.

### 

**main/**

- `assets`: Pasta contendo as imagens e CSS utilizados no projeto;
- `base_dados`: Pasta contendo os CSVs com os dados utilizados no projeto;
- `pages`: Pasta contendo as páginas do aplicativo do Streamlit;
- `tabs`: Pasta contendo as tabs que vão nas páginas do aplicativo do Streamlit;
- `util`: Pasta contendo a definição de layout utilizada no projeto;
- `main.py`: Arquivo principal do aplicativo do Streamlit;
- `modelo_sugestao.py`: Script contendo funções e modelos utilizados na previsão;
- `requirements.txt`: Arquivo com todas as dependências necessárias para o projeto.
